## :rocket: Executando

```bash
# Clone o Repositório
$ git clone git@github.com:DavilsonJunior/Desafio-08-bootcamp-rocketseat.git
```

```bash
# Acesse a pasta do projeto e baixe as dependências
$ yarn
```

```bash
# Para verificar os testes use:
$ yarn test
```

```bash
# Rodando o server
$ yarn start
```
